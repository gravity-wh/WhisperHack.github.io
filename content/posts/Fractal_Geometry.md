# From fractal geometry to random fractals

今天听了fiels获得者Hugo Duminil-Copin的汇报，第一次详细了解了随机分形的相关工作。此文作为一个简短的记录总结。

## 1.分形几何基础

### 1.1 自相似性与分数维

相较于传统欧几里得处理的整数维对象，Benoit Mandelbrot 提出的分形理论引入了分数维的概念，用来描述那些无论放大多少倍都表现出相似结构的粗糙形状。

- 自相似性（self-similarity）：局部结构在某种缩放比例下与整体相似。
- 豪斯多夫维数（Hausdorff Dimension，$ d_H$）：超越拓扑维数的度量。例如，Koch Snowflake的拓扑维数是1，但$ d_H$≈1.26，意味着比线更“填充”空间，但又非二维的面。

### 1.2 确定性分形 & 随机分形

- 确定性分形：如谢尔宾斯基三角形，生成规则是固定的。
- 随机分形：生成规则引入了概率。如自然界的海岸线、布朗运动轨迹。

## 2.统计物理中的相变与几何

Hugo 的工作主要集中在**统计力学的数学基础。** 在这里分形是物质在临界点的一种存在状态。

### 2.1 晶格模型

物理学家用简化的晶格来模拟原子的自旋或相互作用。

* **伊辛模型 (Ising Model)** : 模拟铁磁性，每个格点有自旋 $\uparrow$ 或 $\downarrow$。
* **渗流模型 (Percolation Model)** : 模拟多孔介质中的流体流动。每个格点或边以概率 $p$ 连通（“打开”），以 $1-p$ 阻断（“关闭”）。

### 2.2 相变与临界性

当参数（如温度 $T$ 或概率 $p$）变化时，系统宏观性质会发生突变。

* $p < p_c$ **(亚临界)** : 只有小的连通簇。
* $p > p_c$ **(超临界)** : 出现一个无限大的连通簇（巨型簇）。
* $p = p_c$ **(临界点)** : 这是最神奇的时刻。此时，连通簇的大小分布遵循幂律（Power Law），且这些簇的 **边界呈现出分形结构** 。

## 3. Hugo Duminil-Copin 的议题

Hugo Duminil-Copin 因其在**概率论**和**统计物理**尤其是**三维及以上维度的相变理论**上的突破性贡献而获得菲尔兹奖。

### 3.1 相变的锐利性 (Sharpness of Phase Transition)

在渗流理论中，一个长期存在的问题是：相变是平滑发生的，还是瞬间发生的？
Duminil-Copin 证明了在广泛的模型中，相变是“锐利”的。这意味着对于 $p < p_c$，连通簇的大小呈指数级衰减；一旦跨过 $p_c$，宏观行为迅速改变。

### 3.2 临界点的共形不变性 (Conformal Invariance)

在二维平面上，物理学家长期猜想：在临界点 $p_c$，系统的几何性质在 **共形变换** （保持角度不变的变换，如缩放、旋转）下是不变的。

* 这意味着如果你在临界点观察一个渗流簇的边界，它的**分形维数**是固定的，且具有某种旋转对称性。
* Duminil-Copin 与合作者利用离散全纯函数（Discrete Holomorphic Functions）等工具，严格证明了某些晶格模型（如伊辛模型）在缩放极限下的共形不变性。这连接了离散数学与复分析。

### 3.3 连续性与不连续性

他解决了一个关于 Potts 模型（Ising 模型的推广）的长期猜想，确定了在不同维度和状态数下，相变何时是连续的（二阶相变），何时是不连续的（一阶相变）。

## 4. 为什么这是一门“分形”的学问？

尽管 Duminil-Copin 的论文充满了概率公式，但其几何直观在于 **Scaling Limits（缩放极限）** 。

当我们让晶格的网格间距趋于零（$\delta \to 0$）时，那些随机的、锯齿状的路径（如畴壁、渗流簇边界）收敛为什么？

* 它们收敛为 **随机分形曲线** 。
* 这些曲线可以通过 **Schramm-Loewner Evolution (SLE)** 来描述。SLE 是一类描述二维临界系统缩放极限的随机曲线演化方程。

**Duminil-Copin 的工作为这些分形曲线的存在及其性质提供了严格的数学证明。** 例如，通过他的工作，我们可以更精确地理解为什么二维临界渗流簇的边界的分形维数是 $4/3$。

## 5. 总结

Hugo Duminil-Copin 的工作处于概率论、组合数学和数学物理的交叉点。

* **传统分形**关注静态的几何描述。
* **Duminil-Copin 的工作**关注**动态的相变过程**中涌现出的几何结构。

理解他的工作，有助于我们理解自然界中广泛存在的临界现象——从磁铁的磁化、液体的渗透，到森林火灾的蔓延模型。在这些现象的背后，都隐藏着普适的随机分形法则。
